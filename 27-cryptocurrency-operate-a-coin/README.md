# BlockChain / Cryptocurrency

## Run a proof of concept blockchain cryptocurrency

## Requirements

```
$ git clone https://github.com/cosme12/SimpleCoin.git
$ pip install Flask
$ pip install ecdsa
$ pip install requests
```

## Running it

```
$ cd SimpleCoin
$ python miner.py
```

or

```
$ cd SimpleCoin
$ python wallet.py
```
