# Optical Character Recognition

## Read text in images

## Requirements

```
$ pip install pytesseract
$ pip install pillow
```

Also, install [Tesseract](https://github.com/tesseract-ocr/tesseract/wiki). PyTesseract is just a wrapper around Tesseract.

## Running it

```
$ python pytesseract_snake.py
```
